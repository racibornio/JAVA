1. Download Apache Tomcat - install in 'C:\Program Files\Apache Tomcat\webapps'.
2. Create sub-folder named your app name.
3. Inside of that sub-folder create 'index.jsp' file.
4. Put in the JAVA code that wraps HTML code.
5. In 'C:\Program Files\Apache Tomcat\bin' run the server - 'startup.bat' file.
6. Opn your app on port 8080: 'http://localhost:8080/FirstJSPApp/index.jsp' in your browser.
7. To stop the app run 'shutdown.bat' file in the path from 5th step.
